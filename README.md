# spring-boot-Alura-1
Este é um repositório dedicado ao Curso da Alura de Spring Boot Inicial.

## Objetivo
- Criar uma API Rest;
- Desenvolver um CRUD;
- Fazer Validações;
- Conceito de Paginação e Ordenação;

## Tecnologias que utilizarei:
- Spring Boot 3.4.1;
- Java 17;
- Lombok;
- MySQL / Flyway;
- JPA / Hibernate;
- Maven
- Insomnia;

## Projeto
- Aplicativo mobile para gerenciamento de uma clínica médica.
